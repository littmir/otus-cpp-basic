# Базовый синтаксис 02
## Объявление и определение переменных

#### Объявление переменной
```C++
extern int some_extern_var; // Объявление переменной
```
Когда есть только объявление, переменной можно пользоваться, но компилятор не выделит места под такую переменную, компилятор будет искать именно определение переменной, и будет выдавать ошибки линковки.

#### Определение переменной
*Определение переменной* - ассоциация некоторого адреса в оперативной памяти некоторому имени в программе. Компилятор выбирает смещение относительно начального адреса, а операционная система - начальный адрес.   
*Тип переменной* - размер ячейки памяти.
```C++
int some_global_var; // Определение переменной без инициализации

int some_other_global_var {42}; // Определение переменной с инициализацией
```
Чаще всего переменные *определяются*, а не *объявляются*.

## Инициализация переменных
```C++
unsigned int var = -42; // Инициализация переменной через равенство (ошибки нет, в результате число 4294967290)

unsigned int one_more_var {-42}; // Инициализация через фигурные скобки (ошибка, отрицательное в беззнаковое)
```
Инициализация через *фигурные скобки* более безопасна, не позволяет провести неверные операции, приветствуется в множестве style-guide'ов.

## Пространство имён
```C++
namespace algo {
  // Первое определение пространства имён algo
}

// some code

namespace algo {
  // Второе определение пространства имён algo, что является расширением пространства имён
}
```
В языке С++ таким образом можно продлевать только пространства имён.
