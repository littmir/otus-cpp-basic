# ООП. Абстракция и инкапсуляция

## Static
Статическими могут быть как **поля**, так и **функции-члены** класса.  

Если переменная в классе статическая, то она существует вне объекта класса, она может быть только одной на все возможные объекты, к ней можно обращаться снаружи класса.  

Статический метод может быть также доступен вне объектов классов. Из таких методов можно обращаться к статическим полям класса.  

К статическим полям можно обратиться через объекты класса (через точку), но это плохой стиль, и всегда нужно обращаться только через имя класса (через двойное двоеточие).  

## Class

**Класс** - это правило, по которому рождаются объекты (чертеж).  

## Const

Функции можно перегружать по константности.  

Константность функции - это контракт.  

Если объект класса передается *по константной ссылке*, то вызывать можно только *константные методы*.  

## Mutable

Ключевое слово **mutable** отменяет **const**.  

Используется для изменения константных объектов при этом сохранив видимое извне состояние константности этих объектов.  

Mutable для полей позволяет менять их значение даже в константных методах.  

## Friend

Ключевое слово **friend** отменяет **private**.  

Friend может быть функция или класс.

Friend позволяет иметь доступ к приватным полям класса.
