## Ссылки
*Ссылка* - втрое имя переменной.  
Ссылка реализована внутри через указатель. Не может быть нулевой, что позволяет избежать множества ошибок. Гарантировано, что ссылка имеет значение. Компилятор эффективно работает с ссылками. Ссылка является Zero Cost Abstraction - бесплатная абстракция, когда на что-то стандартное добавили что-то ещё, например как со ссылкой - на указатель добавили условие не пустого значения.

## Входные значения функции по умолчанию

У функции могут быть значения по умолчанию.
```C++
void
sum(int a, int b, int c = 42) {
  // some code here
}

int
main() {
  sum(1, 2, 3); // Результат 6
  sum(1, 2) // Результат 45
}
```
Однако, не может быть случая, когда, например, после значения по умолчанию, идёт неопределенное значение, а потом снова значение по умолчанию. Так нет однозначности при вызове функции с двумя аргументами.
```C++
void
sum(int a = 1, int b, int c = 42) {} 
int
main() {
  sum(1, 2, 3); // Результат 6
  sum(1, 2) // // Ошибка
}
```
